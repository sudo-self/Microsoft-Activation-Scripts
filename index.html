Windows Activator

Scan to download the activation script

Microsoft Product Keys here. Copy the correct key that matches the windows version your activating (example: Windows 10 Pro).

First, open CMD (Command Prompt) as Administrator using:

+ R

Then type cmd.exe and press Enter.

Run these commands one at a time:

slmgr /ipk -your product key here-
slmgr /skms kms8.msguides.com
slmgr /ato

Note: After running, click OK.
